# Graficacion_Pygame2_Castro_Flores_Angel_Johan
Codigos de Programas Básicos usando Pygame 

Sesion4_ej1.py: El resultado de este codigo es una pelota azul que rebota entre los bordes izquierdo y derecho de la ventana. Con cada rebote, la pelota acelera y se mueve más rápido que en el ciclo anterior.

Sesion4_ej2.py: Este código en Pygame crea una ventana y dibuja un círculo azul que pulsa o late, es decir, su radio cambia de tamaño de forma continua. La posición del círculo se mantiene fija en el centro de la pantalla.

Sesion4_ej3.py: Este código simula el movimiento de una pelota que cae y rebota en el fondo de la pantalla, con un efecto de pérdida de energía en cada rebote. Es una simulación simple de la gravedad.

Sesion5_ej1.py: Este código en Pygame carga una imagen y permite al usuario redimensionarla de forma interactiva usando las teclas + y -. La imagen redimensionada siempre se mantiene centrada en la ventana.

Sesion5_ej2.py: Este código implementa una animación de un sprite sheet, creando una figura que parece estar en movimiento constante en el centro de la pantalla.

Sesion5.mini.py: Este código simula el control de una nave espacial que puede girar para seguir la posición del mouse y moverse en la dirección en la que apunta cuando se presiona la tecla W. Es un ejemplo clásico de movimiento basado en trigonometría en videojuegos.

Sesion6_ej1.py: El usuario controla un cuadrado azul que se mueve con las teclas WASD. Si el cuadrado azul toca el cuadrado rojo, su color cambia a verde y no puede cruzar o entrar al área del cuadrado rojo, forzando una interacción de límite/bloqueo.

Sesion6_ej2.py: Este código implementa un juego simple de recolección de objetos, donde el jugador (un cuadrado azul) debe moverse para "recoger" un círculo rojo. Cada vez que el jugador recoge el círculo, el contador aumenta y el círculo aparece en una nueva posición aleatoria. El jugador solo se puede mover con las teclas WASD

Sesion6_ej3.py: Este código implementa un juego de esquivar obstáculos donde el jugador (un cuadrado azul) debe moverse para evitar chocar con múltiples círculos rojos que se mueven horizontalmente. Si el jugador colisiona, el juego se reinicia. El jugador solo se puede mover con las teclas WASD

Sesionmini.py: Este código implementa un minijuego de esquivar y recolectar con una nave espacial. El jugador controla la nave con WASD, la nave gira para reflejar la dirección del movimiento, debe esquivar obstáculos (círculos naranjas) y recolectar objetos (círculos rojos) para ganar puntos.
